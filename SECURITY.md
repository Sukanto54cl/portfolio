# Security Policy

## Supported Versions

The following versions of this project are currently supported with security updates:

| Version | Supported |
| ------- | --------- |
| 5.1.x   | ✅ Yes    |
| 5.0.x   | ❌ No     |
| 4.0.x   | ✅ Yes    |
| < 4.0   | ❌ No     |

Only the versions marked as supported will receive security updates.
Users are encouraged to upgrade to a supported version to remain secure.

## Reporting a Vulnerability

Security issues are taken seriously and responsibly handled.

### How to Report

If you discover a security vulnerability, please report it using one of the following methods:

- Email: sukantodas1993@gmail.com

⚠️ Please do not disclose security issues publicly through GitHub issues,
pull requests, or discussions.

### What to Expect

- Acknowledgement of your report within 72 hours
- Review and assessment of the reported issue
- Periodic status updates during the investigation
- Resolution through a fix or an explanation if the report is declined

Please keep all vulnerability details confidential until a fix has been released
or disclosure has been coordinated.

Thank you for helping keep this project secure.
